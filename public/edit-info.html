Edit Info Page

Cards must be unpublished in order to edit. Unpublished cards have a red border.

If the card is published you can make it editable by clicking on the Make Editable button.

Making it editable removes the card from the published list and updates the timestamp.

Click on any of the content in the card to turn on edit mode

You can now edit the content or the social icons link. Click on Save or Cancel buttons to save or discard your edits.

HTML Markup for cards

Basic card with image

Images should be 328px wide by 200px high. Images are scaled to fit the width of the card and the max height is set to 200px.

Headers are put in <h2> tags or you can add the h2 class to the element for inline styling. They are bolded and the color is FSL orange.

Text can be in <p> <span> <div> tags.

Example 1

Code for card:
<img src="//placehold.it/328x200" alt="Image title here"><h2>Automotive Card</h2>Here is some text here </p>

Example 2

Code for card:
<img src="//placehold.it/328x200" title="image title here"><p><span class="h2">Tech Sessions</h2> <p>Lorem ipsum socum rocum tootee frootee </p>

Centered Text

To center text within the card add the class block-of-test to the content container.

Example 3

Code for card:
<section class="block-of-text"><div><h2 class="center">Centered Text</h2><p class="center">This should be centered on the page</p></div></section>
